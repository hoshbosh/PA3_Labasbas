COMPILING/RUNNING ON LINUX-SERVER
1. Run 'javac musicServer.java'.
2. Run 'java musicServer [PORT_NUMBER]' where PORT_NUMBER
   is the port number you want the server socket to run on.

COMPILING/RUNNING ON LINUX-CLIENT
1. Run 'javac musicClient.java'.
2. Run 'java musicClient [SERVER_ADDRESS] [PORT_NUMBER]' where PORT_NUMBER
   is the port number you want the server socket to run on, and SERVER_ADDRESS
   is the address of the server.
